# YET-ANOTHER-YOGA-LAYOUT-FORK

## Features

- build with `emscripten`
- sync/async
- wasm/asm
- latest `yoga-layout` with `flex-gap`

## Getting Started

```bash
npm install jeetiss/yet-another-yoga-layout-fork
```

```js
import initYoga from 'yet-another-yoga-layout-fork/asmSync'

const Yoga = initYoga()
```
